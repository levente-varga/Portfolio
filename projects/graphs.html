About

Developed in C# using WinForms

Features a force-directed graph drawing algorithm

Uses 4 graph-generator models (discussed below)

Visualises the graphs with a graph view and three charts

Calculates the average degree distribution, supported by sample generation

Start of development: 2022

Release date (v0.1.0): 29 Apr 2024

Models
Erdős-Rényi model

Connects nodes with a fixed probability.

Degrees follow the Poisson distribution.

Nodes: 100, Probability: 10%

Watts-Strogatz model

Creates small-world networks.

Low average path lengths.

Nodes: 100, Probability: 1%, Mean: 20

Barabási-Albert model

Creates network-like graphs.

New nodes form edges based on degree.

Nodes: 100, Power: 1.55

Bianconi-Barabási model

Similar to Barabási-Albert, with an extra fitness weight.

Nodes: 100, Fitness: 0.95

Icon set

Self-made vector graphic icons in Figma

Necessary for symbolising specific features like gradient edges and automatic graph layout

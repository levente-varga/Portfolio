Project:
JSON PARSER
About

I decided to implement my own because I simply could not find a simple parser in C++ that is not an overkill for small projects

It was also a good practice

Developed in C++

Parses any JSON that follows the RFC4627 standard

Builds on the std::shared_ptr class

Can be pretty-printed

Lightweight, short and easy to use code

Start of development: Sep 2023

auto parser = new Json::Parser();

auto json = parser.parse("test.json");

Parsing

Parsing a JSON file couldn't be simpler. The parse() method returns the root node.

auto node = json->at("users")->at(0)->at("age");

Traverse

Using at() with a key or index parameter on a node returns another Json::Node class.

int age = node->getAs<int>();

Convert

Each node can store one of seven possible types: bool, int, double, std::string, Json::List, Json::Object or std::nullptr_t (for null values).

for (auto user : json->at("users"))

{

    auto node = user->at("name");

    string name = node->getAs<string>();

}

Iterate

Nodes with Json::List and Json::Object types can be iterated through.

About

A cross-platform app that communicates with the taximeter unit via bluetooth

Contract from Ecometer Kft.

Developed in Dart using Flutter for iOS and Android

Full UI design in Figma

Utilises the BLoC pattern

Uses Bluetooth API and RESTful APIs

Start of development: May 2024

Release date: coming soon
